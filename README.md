# 📧 MailBag Webmail Client

A full-stack Gmail-style webmail client that allows users to browse mailboxes, read messages, search emails, and send new emails through a custom-built Node.js mail server.

## 🚀 Features
- View mailboxes
- Read emails
- Search emails
- Compose and send emails
- Delete messages
- Async communication using fetch (AJAX)

## 🛠️ Tech Stack
Frontend: TypeScript, HTML, CSS, Webpack  
Backend: Node.js, Express, IMAP, SMTP, Nodemailer  

## ▶️ How to Run

### Server
cd server/PracticeM6_MailBagServer  
npm install  
npm run build  
npm start  

### Client
cd client  
npm install  
npm run dev  

Open http://localhost:3002

## 🔐 Note
Use Gmail App Password. Do NOT upload credentials.

## 📌 Description
A Gmail-style webmail client built with TypeScript, Webpack, and a Node.js mail server.
